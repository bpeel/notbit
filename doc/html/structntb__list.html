Detailed Description

NtbList - linked list.

The list head is of "NtbList" type, and must be initialized using ntb_list_init(). All entries in the list must be of the same type. The item type must have a "NtbList" member. This member will be initialized by ntb_list_insert(). There is no need to call ntb_list_init() on the individual item. To query if the list is empty in O(1), use ntb_list_empty().

Let's call the list reference "NtbList foo_list", the item type as "item_t", and the item member as "NtbList link". The following code

The following code will initialize a list: 

 ntb_list_init (foo_list);
 ntb_list_insert (foo_list, item1);      Pushes item1 at the head
 ntb_list_insert (foo_list, item2);      Pushes item2 at the head
 ntb_list_insert (item2, item3);         Pushes item3 after item2

The list now looks like [item2, item3, item1]

Will iterate the list in ascending order: 

 item_t *item;
 ntb_list_for_each(item, foo_list, link) {
         Do_something_with_item(item);
 }
